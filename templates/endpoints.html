API Documentation

List of possible endpoints

Get an abbreviated version of the transmitted URL
  • POST / — The method accepts the URL string for shortening in the request body and returns a response with the code 201.
JSON data: 
{
"url": "https://my-long-url.com"
}
Response: 
{
"short_id": "id of your link",
"short_url": "https://shortened-url.com"
}
Return the original URL
  • GET /{short_id} — The method takes the identifier of the shortened URL as a parameter and returns a response with the code 307 and the original URL in the Location header.
